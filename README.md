# Welcome to India Club @ GT 🇮🇳

Namaste! You've found the source code for the [India Club at Georgia Tech](https://indiaclubgt.com) website.

India Club is Georgia Tech's home for Indian culture on campus — we host Garba, Diwali, Holi, and everything in between, and we welcome anyone who wants to celebrate with us.
